# SSSALTY — Shopify T-Shirt Brand

## Überblick
SSSALTY ist eine T-Shirt Marke auf Shopify mit 3 Kollektionen:
- **memez** — Meme-basierte Designs (Facebook, TikTok, Pinterest)
- **hustle** — Motivations-/Hustle-Kultur (TikTok, Instagram)
- **core** — Core-Ästhetik (TikTok, Pinterest)

## Workflows

### 1. Neue Kollektion / Design hinzufügen
1. Design-Assets vorbereiten (SVG/PNG, transparent background)
2. Shopify Admin → Produkte → Produkt hinzufügen
3. Varianten: S/M/L/XL/XXL, Farben
4. SEO: Title, Description, Tags
5. Kollektion zuweisen

### 2. Social Media Posting
- **memez**: Facebook (Memes), TikTok (Short Clips), Pinterest (Pins)
- **hustle**: TikTok (Motivation), Instagram (Reels/Posts)
- **core**: TikTok (Ästhetik), Pinterest (Moodboards)

### 3. Analytics Check
- Shopify Dashboard: Verkäufe pro Kollektion
- Social Media: Engagement-Raten pro Plattform
- Best-Seller identifizieren und nachproduzieren

## Nützliche Befehle
```bash
# Shopify Admin (Browser)
# Social Media Manager (Browser)
```

## Pitfalls
- Keine überlappenden Designs zwischen Kollektionen
- memez = humorvoll, hustle = ernst/motivierend, core = minimalistisch
- Pinterest braucht vertikale Formate (2:3)
- TikTok braucht 9:16 oder 1:1
