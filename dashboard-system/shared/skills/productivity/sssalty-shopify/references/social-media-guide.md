# SSSALTY Social Media Posting Guide

## Plattform-Formate
- **TikTok**: 9:16 Video, max 3 Min, deutsche Untertitel
- **Instagram**: 1:1 Feed Posts, 9:16 Reels
- **Facebook**: 16:9 oder 1:1, Text + Bild
- **Pinterest**: 2:3 vertikale Pins, 1000x1500px empfohlen

## Kollektions-Töne
| Kollektion | Ton | Zielgruppe | Beste Plattform |
|------------|-----|------------|-----------------|
| memez | Lustig, meme-haft | Gen Z, 16-25 | TikTok, Facebook |
| hustle | Motivierend, hart | Young Adults, 20-35 | TikTok, Instagram |
| core | Minimalistisch, ästhetisch | Creatives, 18-30 | TikTok, Pinterest |

## Posting-Frequenz (empfohlen)
- TikTok: 1-2x täglich
- Instagram: 3-5x pro Woche
- Facebook: 2-3x pro Woche
- Pinterest: 5-10 Pins pro Tag
